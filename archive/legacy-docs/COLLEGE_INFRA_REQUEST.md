# College Handover Request

Send this short list to the college team.

## 1) Who owns it?
- Billing owner
- Technical owner
- Backup contact

## 2) Where will it run?
- AWS account or server access
- Region
- Ubuntu server details
- SSH key access
- Public IP or DNS

## 3) Database details
- PostgreSQL database URL or endpoint
- Database name
- Username and password
- Backup and restore plan

## 4) Email details
- SMTP host
- SMTP port
- SMTP username
- SMTP password
- Verified sender email

## 5) Payment details
- Zwitch live API key
- Zwitch live API secret
- Zwitch layer access key if needed
- Confirm live payment endpoint details

## 6) Website URLs
- Frontend URL
- Backend API URL
- Allowed frontend domains

## 7) Security and scale
- Expected number of users
- Expected file uploads/data size
- Required uptime
- Who handles SSL/HTTPS
- Who handles backups and rollback

## 8) Final approval
- Confirm they want the source code kept private with you
- Confirm who can access the repo
- Confirm handover date
